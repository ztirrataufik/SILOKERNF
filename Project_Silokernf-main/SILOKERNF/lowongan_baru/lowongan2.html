PT Indah Gilang Tama

Dibutuhkan Web Developer
PT Indah Gilang Tama melalui program rekrutmennya saat ini sedang membuka lowongan kerja untuk posisi Web Developer di Bekasi yang bertujuan untuk meningkatkan kinerja operasional di dalam maupun luar kantor PT Indah Gilang Tama. Perusahaan Tersebut sedang mencari calon tenaga kerja yang siap diterjunkan ke setiap divisi bagian perusahaan yang sesuai dengan posisi, kemampuan dan keahlian para pencari kerja tersebut. Berikut ini adalah detail lengkap persyaratan untuk posisi Web Developer di Bekasi Perusahaan PT Indah Gilang Tama:
Tugas / Tanggung Jawab :

  • Melakukan Kegiatan dan Pekerjaan Web Developer
  • Menganalisis program dan spesifikasi desain aplikasi Web Programming
  • Mengembangkan aplikasi Web Programming sesuai kebutuhan
  • Berdomisili di Bekasi
Kualifikasi / Persyaratan :
  • Minimal D3/S1 Teknik Informatika/Ilmu Komputer/Sistem Informasi
  • Pria/Wanita maksimal 30 th
  • Bekerja Penuh Waktu
  • Pengalaman di bidang WEB Application Programming
  • Menguasai MS SQL Server, Crystal Report
  • Wajib Menguasai Web Programming (PHP, Javascript, CSS, Jquery, AJAX)
  • Logika kuat dalam query SQL dan Stored Procedure SQL
  • Mampu bekerja sesuai dengan deadline
  • Fresh graduate are welcome
